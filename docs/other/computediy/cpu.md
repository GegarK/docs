# CPU

CPU（Central Processing Unit）：中央处理器，是计算机的大脑，负责处理所有指令、逻辑运算、控制程序执行等核心任务。

## [Intel CPU](https://www.intel.com)

| 系列 | 适用场景 | 特点 |
|------|----------|------|
| **i3** | 基础办公、网页浏览 | 低功耗、价格便宜、核心线程数较少 |
| **i5** | 主流游戏、轻量剪辑 | 性能均衡、性价比高 |
| **i7** | 游戏直播、专业软件 | 高主频、多线程 |
| **i9** | 渲染、视频制作、重度游戏 | 顶级性能、多核心高线程 |
