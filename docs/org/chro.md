# CHRO

## 一、定义

**CHRO（Chief Human Resources Officer，首席人力资源官）** 是公司中**负责整体人力资源战略、组织发展和人才管理**的高管。  
CHRO 不直接创造业务价值，而是通过**优化人才、组织结构和企业文化**来提升公司整体执行力和竞争力。

> 简单理解：  
> **CHRO = 对公司人才、组织和企业文化负最终责任的人**

## 二、核心职责

1. **人才招聘与管理**
   - 制定公司招聘战略，吸引核心人才  
   - 领导招聘团队进行选拔与面试  
   - 优化人才配置，确保关键岗位有人才储备

2. **绩效管理与激励**
   - 建立绩效考核体系（KPI / OKR）  
   - 设计薪酬、奖金、股权激励方案  
   - 激励员工提高工作效率和积极性

3. **组织发展与规划**
   - 设计公司组织架构，明确汇报关系和职责  
   - 推动部门协作与跨部门沟通  
   - 支持公司战略的落地和组织变革

4. **企业文化与员工体验**
   - 塑造公司使命、愿景和核心价值观  
   - 推动企业文化建设和员工认同感  
   - 管理员工满意度、留存率和职业发展规划

5. **人力资源政策与合规**
   - 制定 HR 政策和流程  
   - 确保公司人力管理符合劳动法规和合规要求  
   - 管理劳动争议和员工关系风险

## 三、关键能力

- **战略思维**：将人力资源策略与公司战略紧密结合  
- **组织管理能力**：设计高效组织结构和流程  
- **人才管理能力**：招聘、培养、激励和留住核心人才  
- **领导力与影响力**：领导 HR 团队，推动公司文化落地  
- **沟通与谈判能力**：协调跨部门合作，解决员工关系问题

## 四、职业路径（典型）


> 在创业型公司中，CHRO 往往同时负责招聘、企业文化和组织建设工作。

## 五、总结

- **CHRO 是公司人力资源和组织发展的最终负责人**  
- **负责人才管理、组织架构、绩效激励和企业文化建设**  
- **通过优化人力资源和组织能力，为公司战略落地提供支撑**
