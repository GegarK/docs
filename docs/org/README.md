# ORG

**ORG（Organization / 公司 / 团队的组织结构图或组织架构）** 是企业或团队用来描述**内部管理层级、部门分工与汇报关系**的通用概念。它不仅展示了“谁负责什么”，也帮助明确决策链与职责边界。

```
董事会 / Board of Directors
 └─ CEO（Chief Executive Officer，首席执行官）
     ├─ COO（Chief Operating Officer，首席运营官）
     │   ├─ 业务运营团队
     │   ├─ 客服 / 支持体系
     │   └─ 交付 / 供应链（如有）
     │
     ├─ CFO（Chief Financial Officer，首席财务官）
     │   ├─ 财务管理
     │   ├─ 预算与成本控制
     │   ├─ 融资 / 投资者关系
     │   └─ 审计 / 税务
     │
     ├─ CTO（Chief Technology Officer，首席技术官）
     │   ├─ 技术架构与研发方向
     │   ├─ 研发团队（后端 / 前端 / 客户端）
     │   ├─ 基础设施 / 平台工程
     │   └─ CISO（Chief Information Security Officer，首席信息安全官）
     │       ├─ 应用安全 / 云安全
     │       ├─ SOC / 安全运营
     │       ├─ 漏洞管理 / 攻防
     │       └─ 数据与隐私保护
     │
     ├─ CSO（Chief Security Officer，首席安全官）
     │   ├─ 企业级安全战略
     │   ├─ 风险管理（Risk / GRC）
     │   ├─ 合规与审计（ISO / SOC2 / 等保）
     │   └─ 重大安全事件最终决策
     │
     ├─ CIO（Chief Information Officer，首席信息官）
     │   ├─ 内部 IT 系统
     │   ├─ OA / ERP / 内部平台
     │   └─ IT 运维与支持
     │
     ├─ CPO（Chief Product Officer，首席产品官）
     │   ├─ 产品战略与路线图
     │   ├─ 产品经理团队
     │   └─ 用户体验 / 增长
     │
     └─ CHRO（Chief Human Resources Officer，首席人力资源官）
         ├─ 招聘与组织发展
         ├─ 绩效与激励
         └─ 企业文化
```
